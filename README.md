# Python Study Repository

이 레포지토리는 **"혼자 공부하는 파이썬"** 책을 기반으로 한 파이썬 학습 프로젝트 모음입니다. 각 챕터별로 실습 코드와 추가 조사 내용을 포함하고 있습니다.

## 📚 프로젝트 개요

이 레포지토리는 파이썬 데이터 수집, 분석, 시각화에 대한 실습 프로젝트들을 포함합니다. 각 챕터는 책의 내용을 따라 작성되었으며, 추가적인 조사와 실습을 통해 학습 내용을 심화했습니다.

## 📁 챕터별 프로젝트

### [Chapter 07: 웹 스크래핑을 활용한 주식 데이터 수집 및 분석](./Chapter_07/)

Playwright를 사용하여 네이버 금융에서 코스피 시가총액 데이터를 웹 스크래핑하고, 수집한 데이터를 분석하여 트리맵으로 시각화하는 프로젝트입니다.

**주요 학습 내용:**
- Playwright를 활용한 브라우저 자동화
- 동적 웹 페이지에서 테이블 데이터 추출
- 데이터 전처리 및 분석
- Plotly를 활용한 트리맵 시각화

### [Chapter 09: 공공 API를 활용한 경제 지표 데이터 수집 및 시각화](./Chapter_09/)

한국은행 ECOS API를 활용하여 경제 지표 데이터를 수집하고, 수집한 데이터를 시각화하는 프로젝트입니다.

**주요 학습 내용:**
- 한국은행 ECOS API 활용
- 기준금리, 국고채, 회사채, 코스피지수, 원달러환율 등 경제 지표 수집
- Excel 파일로 데이터 저장
- Matplotlib과 Seaborn을 활용한 시계열 데이터 시각화

### [Chapter 10: 금융 API를 활용한 금리 정보 수집 및 문서 생성](./Chapter_10/)

금융감독원 FSS API와 한국은행 ECOS API를 활용하여 금융 상품 정보 및 금리 데이터를 수집하고, Word 문서로 자동 생성하는 프로젝트입니다.

**주요 학습 내용:**
- 금융감독원 FSS API와 ECOS API 활용
- 정기예금, 정기적금, 대출 금리 등 금융 상품 정보 수집
- 금리 추이 그래프 생성
- Python-docx를 활용한 Word 문서 자동 생성
- **미니프로젝트**: 책에서 배운 FSS API와 수업에서 배운 GitHub Actions + crontab을 응용한 채용 공고 자동 업데이트 시스템

### [Chapter 11: 공공 API를 활용한 부동산 데이터 분석 및 지도 시각화](./Chapter_11/)

공공데이터포털의 API를 활용하여 서울특별시의 아파트 매매 실거래가 데이터를 수집하고, 지도 위에 단계 구분도로 시각화하는 프로젝트입니다.

**주요 학습 내용:**
- 공공데이터포털 API 활용
- 서울시 아파트 실거래가 데이터 수집 및 분석
- 자치구별 단위 면적당 평균 매매가 분석
- 거래 빈도 및 가격 변동률 분석
- GeoPandas를 활용한 지도 시각화

## 🛠️ 기술 스택

- **웹 스크래핑**: Playwright
- **데이터 처리**: Pandas
- **시각화**: Matplotlib, Seaborn, Plotly, GeoPandas
- **API 활용**: datakart 라이브러리 (ECOS, FSS, 공공데이터포털)
- **문서 생성**: python-docx
- **자동화**: GitHub Actions

## 📋 공통 설정

대부분의 프로젝트는 API 키가 필요합니다. 각 챕터 폴더에 `.env` 파일을 생성하고 필요한 API 키를 설정하세요:

```env
# Chapter 09, 10
ECOS_KEY="YOUR_ECOS_API_KEY"

# Chapter 10
FSS_KEY="YOUR_FSS_API_KEY"

# Chapter 11
DATAGO_KEY="YOUR_DATAGO_API_KEY"
SGIS_KEY="YOUR_SGIS_API_KEY"
SGIS_SECRET="YOUR_SGIS_SECRET"
```
